The Vision

Moulify is an open-source component library delivered as a Nuxt module. It gives your app a unified set of UI building blocks—buttons, icons, form elements, and more on the way—backed by a consistent design language and a massive icon system. No glue code, no style conflicts, no juggling five different packages. One module, one design system, ready to grow with your product.

Built Different

Moulify is building toward a complete, production-ready component library for the Nuxt ecosystem. Instead of stitching together headless primitives from different authors, you get a single module with a cohesive design language, growing set of components, and an ever-expanding icon system—all built for Vue 3 and Nuxt from the ground up.

  • A growing component library — Starting with foundational pieces like Button and Icon, with more components shipping regularly. The goal: every UI building block your Nuxt app needs, in one package.
  • Built for the Nuxt ecosystem — Not a generic Vue library wrapped in a module. Moulify leverages Nuxt auto-imports, plugins, and module hooks so components just work the moment you add the module.
  • 1,000+ type-safe icons — A Fantasticon-generated icon font with TypeScript union types. Use any icon by name and let your editor catch typos at build time, not in production.
  • Themeable by design — Configure a prefix and color tokens in nuxt.config to match your brand. As the library grows, theming will extend to every component.
  • Zero setup, immediate value — One install, one line in your config. No CSS imports to remember, no global plugin registration, no separate icon package. It just works.
  • Open source & community-driven — MIT licensed, contributions welcome. Shape the library alongside other Nuxt developers.
